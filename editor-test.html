BasicRichTextEditor Test Page

Testing Instructions

Font Functionality Tests:

  1. Select text and try changing font family from the dropdown
  2. Verify that each font option in the dropdown shows in its actual font
  3. Test font size changes
  4. Try bold, italic, underline formatting

Scripture Reference Tests:

  1. Type scripture references like "John 3:16" or "Romans 8:28"
  2. Wait for them to automatically highlight with golden styling
  3. Test that clicking on highlighted references works
  4. Verify that scripture highlighting doesn't interfere with font changes

Integration Tests:

  1. Apply font styles to text that contains scripture references
  2. Add scripture references to already formatted text
  3. Verify both features work together harmoniously
